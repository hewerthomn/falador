# Falador

Falador a partir do terminal, powered by Google Translate


### Usando
* Dê permissão de execução

`chmod +x falador`

* Divirta-se!

  `./falador "Onde está o futuro que nos prometeram?`
  
  `./falador -l en "Luke, I am your father.`
