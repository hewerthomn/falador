# Falador

Falador a partir do terminal, powered by Google Translate


### Usando
* Dê permissão de execução

`chmod +x falador`

* Divirta-se!

  `./falador "Quem roeu a roupa do rei de Roma?`
  
  `./falador -l en "I'm fucking crazy, bitch`
